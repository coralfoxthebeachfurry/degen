## 1. Welcome & Philosophy

Welcome to the team. You've been trusted to keep Coral's Degenerates running smoothly.

**Our philosophy:** Enforce the rules, but don't be a robot. Use judgment. A warning is often better than a ban. But when lines are crossed (CSAM, gore, doxxing), act fast and permanently.

> *"Harmony, not hostility."*

---

## 2. Mod Expectations

| Expectation | Details |
|-------------|---------|
| **Activity** | Be visible at least a few times per week. Ping mod chat if you'll be away >5 days. |
| **Consistency** | Enforce the same way for everyone – friends, regulars, and newbies alike. |
| **Confidentiality** | Do not share mod chat contents, user reports, or internal decisions publicly. |
| **De-escalation** | Be calm. Don't argue with rulebreakers. Issue the action and move on. |
| **Teamwork** | Check mod chat before making big calls (permabans, appeals). |

## 3. Rule-by-Rule Enforcement Guide

### Rule 1 – CSAM / Gore
**Zero tolerance.** No warning. No timeout.
- **Action:** Immediate permaban + report to Discord Trust & Safety.
- **Notify:** Ping @Owner and @Admin in mod chat with screenshot.

### Rule 2 – Heated Arguments / Drama
**Action ladder:**
1. Warn in-channel: *"Take this to DMs please."*
2. If they continue → Timeout (10–30 min)
3. Repeat offender → Kick
4. Continued after kick → Ban

### Rule 3 – NSFW
**Action ladder:**
| Offense | Action |
|---------|--------|
| 1st | Timeout (1 hour) + warning |
| 2nd | 7-day mute or kick |
| 3rd | Ban |

### Rule 4 – Spam / Disruption
**Action:** Mute or timeout (15 min – 1 hour) without prior warning for clear spam.
- Repeat offenses → Kick → Ban

### Rule 5 – Hate Speech / Harassment
**Serious offenses (slurs, threats, doxxing):** Jump to kick or ban immediately.
**Lesser offenses:** Warning → Kick → Ban

### Rule 6 – Channel Relevance
**Action:** Delete off-topic post without warning (first time). Second offense → timeout (30 min). Repeated → kick.

### Rule 7 – Unsolicited DMs
**If reported:**
1. Tell the harasser to stop (mod DM)
2. If they refuse → Warning
3. Continues → Ban

### Rule 9 – Malicious Links / Files
| Offense | Action |
|---------|--------|
| 1st | Warning |
| 2nd | Temp ban (7 days) |
| 3rd | Permaban |

### Rule 10 – Reporting Violations
**Action:** Reminder not to backseat mod. No punishment unless they repeatedly interfere.

## 4. Action Ladder

| Action | What it means | When to use |
|--------|---------------|-------------|
| **Warning** | Verbal or typed reminder | First-time minor offenses |
| **Timeout** | 10min – 24hr mute | Spam, NSFW 1st, ignoring warnings |
| **Kick** | Removes user (they can rejoin) | Repeated minor offenses |
| **Temp Ban** | 1–30 day ban | Malicious links 2nd offense |
| **Permaban** | Permanent removal | CSAM, gore, doxxing, hate speech |

## 6. Escalation Chain

| Issue type | Ping first | If no response (30 min) |
|------------|------------|-------------------------|
| CSAM / Gore | @Owner | @Admin |
| Ban appeal | @Admin | @Owner |
| Permaban decision | Discuss in mod chat | @Owner |
| Unsure about a rule | @Mod (ask team) | @Admin |

## 7. Handling Appeals & Disputes

### If a banned user DMs you:
- **Do not** argue with them
- **Do not** permaban them again
- **Reply with:** *"Please open a ticket in the server or DM @Admin for appeals."*

### If a mod disagrees with your action:
1. Discuss in mod chat privately
2. If still in disagreement, @Admin makes final call
3. No public fighting about mod decisions

---

## 8. What Mods CANNOT Do

| ❌ Cannot | ✅ Instead |
|-----------|------------|
| Demod other mods | Ping @Admin |
| Ban someone for personal reasons | Ask another mod to handle if biased |
| Ignore reports from users you dislike | Treat all reports equally |
| Leak mod chat | Keep internal discussions internal |
| Threaten users ("I'll ban you if...") | Just take the action calmly |

## 10. DM Response Templates

### Warning (minor offense)
> *"Hey @user, just a heads up – {rule violated} isn't allowed here. Please keep it in mind going forward. Thanks!"*

### Timeout notification
> *"@{user}, you've been timed out for {duration} due to {reason}. Feel free to continue participating after the timeout ends."*

### Ban notification (user DMs you)
> *"You've been banned from Coral's Degenerates for violating {rule}. This decision is final. Please do not attempt to evade the ban."*

### Responding to a report (from a user)
> *"Thanks for the report. We're looking into it."*

## 11. Logging & Audit Trail

All moderation actions must be logged in `#audit-log` for transparency and team reference.

### What to Log

- **Warning** - User @, rule violated, date, your name
- **Timeout / Mute** - User @, duration, reason, date
- **Kick** - User @, reason, date, screenshot (if available)
- **Ban (temp or perm)** - User @, reason, duration (if temp), evidence link
- **Appeal decision** - User @, original violation, decision, who decided

### Log Format Template

Use this format for every entry:

`[ACTION] username (User ID [UID])
Reason: [reason]
Breaks: [CoC section and subsection/rules]
Duration: [if applicable]
Additional: [if any]
Date: YYYY-MM-DD`

### Examples

`Timed out and warned fluffycat (User ID [UID])
Reason: NSFW in chat
Breaks: CoC section 3 / Rule 3
Duration: 1 hour
Date: 2026-04-26`

`Timed out louduser (User ID [UID])
Reason: Spam / copy-pasta
Breaks: CoC section 4 / Rule 4             
Duration: 30 minutes         
Date: 2026-04-26`

`Banned badactor (User ID [UID])
Reason: CSAM
Breaks: CoC section 1 / Rule 1             
Duration: Permanent         
Date: 2026-04-26`

### Screenshots & Evidence

- Always screenshot the violating message before deleting it.
- Post the screenshot in `#audit-log` alongside the log entry.
- For DM harassment reports, ask the reporter to screenshot and send to mods.

### Do NOT Log Here

- Private user appeals (keep those in mod chat or tickets)
- Internal mod disagreements
- Speculative notes about users (only confirmed violations)

### Audit Log Maintenance

- Keep `#audit-log` viewable by non-mods but only postable by mods
- Pin the latest 5-10 important entries
- Older logs stay in chat history

---

**Why logging matters:** When a user appeals or repeats an offense, you need to see their full history. A clean audit log protects both the user and the mod team.
